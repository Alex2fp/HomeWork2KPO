# HSE Bank — Модуль "Учёт финансов" (Console, .NET)

## Цель
Реализовать доменную модель и консольное приложение для управления личными финансами:
- Счета (BankAccount)
- Категории (Category)
- Операции (Operation: доход/расход)


## Реализованные требования (там, где это возможно и уместно)
- Создание/редактирование/удаление счетов, категорий, операций.
- Аналитика: баланс счёта, сумма доходов/расходов, разница за период, группировка по категориям.
- Импорт: CSV, JSON, YAML (Template Method).
- Экспорт: CSV, JSON, YAML (Visitor).
- Управление данными: пересчёт баланса (команда RecalculateBalance), валидации инвариантов.
- Статистика: измерение времени выполнения пользовательских сценариев (Decorator).
- Паттерны GoF: **Фасад, Команда, Декоратор, Шаблонный метод, Посетитель, Фабрика**.
- SOLID/GRASP: High Cohesion/Low Coupling, DI-контейнер.
